Simple Stock Visualisations

with % Change, and Recession Plots

Rob Schick, PhD
MASTS Research Fellow, University of St Andrews

Goals

  • Create a Shiny app to:
    • plot a stock based on user input (default to IBM)
    • allow the user to specify the online source (e.g. Google)
    • allow a date range
    • allow the user to overlay the 2008 financial crisis as a rectangle (required addTA from quantmod)
    • Calculate the % change in the stock over the input time

Stand on the Shoulders of Giants
